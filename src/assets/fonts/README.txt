Directorio de fuentes personalizadas

Este directorio está destinado a almacenar fuentes estilo graffiti o pixel art
que se utilizarán en la aplicación Graffiti Vision.

Formato recomendado:
- TTF (TrueType Font)
- OTF (OpenType Font)

Ejemplos de fuentes que encajan con el tema:
- Fuentes estilo graffiti/street art
- Fuentes pixel art/retro
- Fuentes tipo terminal/monospace con estilo

Las fuentes se cargarán dinámicamente en la aplicación para mantener
el estilo visual consistente con el tema graffiti/retro.
